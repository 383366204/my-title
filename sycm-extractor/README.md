# 生意参谋数据提取插件

一键提取 淘宝/天猫 生意参谋搜索分析数据到剪贴板，配合 [my-title](../README.md) 标题生成工具使用，快速完成选品标题创作。

## 功能

- 📊 **自动捕获数据**: 在生意参谋搜索分析页面自动监听数据加载
- 📋 **一键复制导出**: 点击插件图标即可提取全量数据并复制到剪贴板
- 🎯 **完整字段提取**: 支持相关搜索词、搜索人气、点击率、支付转化率、支付买家数、需求供给比、天猫商品点击占比7个核心指标
- 🔄 **配合 AI 标题生成**: 导出的数据可直接导入 my-title 工具进行标题生成

## 安装步骤

1. 下载本项目代码到本地
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`
3. 打开右上角的「开发者模式」开关
4. 点击「加载已解压的扩展程序」
5. 选择 `sycm-extractor` 文件夹完成加载

> 注意：该插件仅用于本地开发，不会上传到 Chrome Web Store

## 使用方法

1. 登录 淘宝/天猫 生意参谋
2. 进入「市场」→「搜索分析」页面
3. 输入你关注的核心关键词，等待搜索结果数据加载完成
4. 确保所有数据已经展示在页面上（需要翻页的请翻完所有页）
5. 点击浏览器工具栏上的插件图标
6. 点击「提取并复制」按钮
7. 数据已经复制到剪贴板，可以粘贴到 Excel 或其他工具中处理

## 输出格式

数据以 **Tab 分隔符** 格式输出，共 7 列：

| 列号 | 字段名 | 说明 |
|------|--------|------|
| 1 | 相关搜索词 | 相关的搜索关键词 |
| 2 | 搜索人气 | 该关键词的搜索人气 |
| 3 | 点击率 | 点击率百分比 |
| 4 | 支付转化率 | 支付转化率百分比 |
| 5 | 支付买家数 | 支付买家数量 |
| 6 | 需求供给比 | 需求供给比指标 |
| 7 | 天猫商品点击占比 | 天猫商品点击占比百分比 |

**示例输出:**
```
连衣裙女	12345	6.2	3.1	450	45	62.5
牛仔短裤男	8765	5.8	2.9	380	38	58.3
```

## 与 my-title MCP 工具配合使用

1. 使用本插件从生意参谋提取搜索词数据
2. 将数据粘贴到文本编辑器，提取第一列的所有搜索词
3. 在支持 MCP 的客户端（如 Claude Desktop、Cursor、OpenClaw）中调用 `generate_title` 工具
4. 逐个关键词生成 SEO 优化的商品标题，批量完成铺货标题创作

## 注意事项

1. **必须等待数据加载完成**: 请确保生意参谋页面已经完全加载所有搜索结果数据后再点击提取
2. **域名支持**: 仅支持 `sycm.taobao.com` 域名
3. **数据完整性**: 如果数据分页，请确保所有页面都已加载，插件会自动收集所有已经加载的数据
4. **格式兼容性**: Tab 分隔格式可以直接粘贴到 Excel、Google Sheets 等表格工具中
5. **隐私安全**: 所有数据提取都在浏览器本地完成，不会上传到任何第三方服务器
